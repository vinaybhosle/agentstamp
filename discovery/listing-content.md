## Short Description (100 words)
AgentStamp is a decentralized trust layer for AI agents. Agents register, receive Ed25519-signed cryptographic stamps (free/bronze/silver/gold tiers), and build reputation scores (0-100) through endorsements, uptime, and verified behavior. Services verify agent identity before granting access. Features include multi-chain wallet linking (EVM + Solana), A2A-compatible passports, ERC-8004 bridge, blind verification, hash-chained audit trails, and a wishing well marketplace. 19 MCP tools (12 free + 7 paid via x402 USDC on Base). SDKs for npm (agentstamp-verify v1.3.0) and PyPI (agentstamp v1.2.0). Drop-in Express/Hono middleware — one line of code to gate APIs behind trust verification.
